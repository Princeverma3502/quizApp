# Quiz App

Minimal React + Vite quiz application implementing the requested features:

- Load quiz
- Display questions and choices
- Show immediate correct/incorrect feedback
- Confetti animation on correct answers
- Results summary at the end

Run locally:

```bash
cd quiz-app
npm install
npm run dev
```

Open the app at the URL shown by Vite (usually http://localhost:5173).

Guidance for committing subtask changes and pushing to your GitHub repo:

1) Make a focused commit for each completed subtask, for example:

```bash
git add .
git commit -m "F01: Add demo questions (subtask)"
git push origin main
```

2) Replace `main` with your working branch name if you use feature branches.
3) Make sure your remote is set to `https://github.com/Princeverma3502/quizApp.git` or your preferred repo URL:

```bash
git remote add origin https://github.com/Princeverma3502/quizApp.git
git push -u origin main
```

Repeat for each subtask, updating the commit message to reflect the subtask (e.g., `F02: Improve UI styles`).
